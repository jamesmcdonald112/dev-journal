Print (@media print)

- @media print {} applies only when printing or in print preview.
    
- Use it to remove UI (nav, buttons, forms, ads).
    
- Switch to black text on white background.
    
- Show full URLs for links if needed.
    
- Prevent awkward page breaks using:
    
    - break-inside: avoid;
        
    - break-after: avoid;
        
    
- Print CSS is for readability and structure, not visual polish.
