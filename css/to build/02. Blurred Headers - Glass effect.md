### **Goal**

  

A **site-wide sticky header** that creates a **glass effect** while scrolling over a hero image.

---

### **Key rules**

- Sticky sticks **within its container**. Put the header in the main page flow if you want it to stick across the whole page.
    
- Glass effect needs:
    
    - **semi-transparent background** (bg-*/xx)
        
    - **backdrop blur** (backdrop-blur-*)
        
    
- Don’t let overflow-hidden wrap the header or it may clip the shadow.
    
- If you use z-*, make the element **positioned** (relative/sticky/fixed/absolute) or it won’t apply.
    

---

### **Steps**

  

#### **1) Create a hero section with a background image (clipped)**

```html
<section class="relative h-[70vh] overflow-hidden">
  <img
    src="/images/lady-justice3.webp"
    class="absolute inset-0 size-full object-cover"
    alt=""
  />

  <!-- Content must be above the image -->
  <div class="relative z-10 p-10">
    <h1 class="text-5xl font-bold text-white">Hero Title</h1>
  </div>
</section>
```

#### **2) Add a sticky header** 

#### **above**

####  **the hero (so it sticks across the page)**

```html
<header class="sticky top-0 z-50">
  <div class="mx-auto max-w-6xl px-6 py-4">
    Header
  </div>
</header>
```

#### **3) Turn it into “glass”**

Use a **transparent background** + **backdrop blur** + **border**:

```html
<header class="sticky top-0 z-50 bg-white/40 backdrop-blur-md border-b border-black/10">
  <div class="mx-auto max-w-6xl px-6 py-4">
    Glass Header
  </div>
</header>
```

#### **4) Make it obvious it’s working (temporary)**

```html
<header class="sticky top-0 z-50 bg-white/20 backdrop-blur-xl border-b border-black/10">
  <div class="mx-auto max-w-6xl px-6 py-4">
    Strong Glass
  </div>
</header>
```

#### **5) Test properly**

- Put a spacer **above** the header so you can see it move before it sticks:
    

```html
<div class="h-40">Spacer above header</div>
```

- Scroll. The header should “hit” the top and then stick.
    
- If blur looks like nothing:
    
    - your background is too opaque (bg-white/80 hides blur)
        
    - or there’s no detailed content behind it (needs image/texture to see blur)
        
    

---

### **Extra challenge: shadow that doesn’t get clipped**

  

Shadow can be clipped if an ancestor has overflow-hidden. Keep header outside clipped containers.

```html
<header class="sticky top-0 z-50 bg-white/30 backdrop-blur-lg border-b border-black/10 shadow-lg">
  <div class="mx-auto max-w-6xl px-6 py-4">
    Glass + Shadow
  </div>
</header>
```
