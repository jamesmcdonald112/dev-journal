### **Goal**

  

A sticky header that blurs the hero image behind it.

  

### **Steps**

1. **Create a hero section with a background image**
    

```
<section class="relative h-[70vh] overflow-hidden">
  <img src="/images/lady-justice3.webp" class="absolute inset-0 size-full object-cover" alt="" />
  <div class="relative z-10 p-10">
    <h1 class="text-5xl font-bold text-white">Hero Title</h1>
  </div>
</section>
```

2. **Add a sticky header above it**
    

```
<header class="sticky top-0 z-50">
  <div class="mx-auto max-w-6xl px-6 py-4">
    Header
  </div>
</header>
```

3. **Turn it into “glass”**
    
    Add these classes to the header:
    

```
<header class="sticky top-0 z-50 bg-white/60 backdrop-blur-md border-b border-black/10">
```

4. **Make it obvious it’s working**
    
    Temporarily increase blur + reduce opacity:
    

```
<header class="sticky top-0 z-50 bg-white/30 backdrop-blur-xl border-b border-black/10">
```

5. **Test**
    

  

- Scroll down.
    
- Watch the image behind the header blur through it.
    
- If you don’t see blur, it’s because the header isn’t over an image or the bg is too opaque.
    

  

### **Extra challenge**

  

Add a shadow that doesn’t get clipped:

```
<header class="sticky top-0 z-50 bg-white/40 backdrop-blur-lg border-b border-black/10 shadow-lg">
```
