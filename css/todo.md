### **1) CSS TODO plan (fits your Scrimba course + builds your “lifetime” files)**

  

**Phase A — Base (1 session)**

- Create /css/reset.css (use your modern reset)
    
- Decide: **reset vs modern-normalize** (pick one, don’t stack)
    
- Add /css/tokens.css (spacing + font sizes + colors as CSS variables)
    

  

**Phase B — Composition (Every Layout) (6–10 short sessions)**

Go one pattern at a time and add it to /css/composition.css:

- Stack
    
- Cluster
    
- Center
    
- Sidebar
    
- Switcher
    
- Reel (optional)
    
- Frame / Cover (optional)
    

  

Use Every Layout’s own generators + examples. 

  

**Phase C — Utilities (2 sessions)**

Create /css/utilities.css with only the “forever” helpers:

- .wrapper / .container
    
- .flow
    
- .sr-only (+ focusable variant if you use skip links)
    
- .list-reset
    
- .visually-hidden (if you prefer that naming)
    
- a few text utils you genuinely reuse (.text-center, .uppercase)
    

  

**Phase D — Blocks (per project)**

Per site: /css/blocks/*.css for actual components

- buttons
    
- cards
    
- nav
    
- form styles
    
- hero
    

  

**Phase E — Exceptions (per block as needed)**

Use state hooks (data-state, aria-*) for variants.
