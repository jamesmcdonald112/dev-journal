## Reset references

- Andy Bell — Modern CSS Reset (blog): https://bell.bz/a-modern-css-reset/
- Andy Bell — Modern CSS Reset (gist): https://gist.github.com/Asjas/4b0736108d56197fce0ec9068145b421
- Eric Meyer — Reset CSS (v2.0): https://meyerweb.com/eric/tools/css/reset/
- Tailwind — Preflight (if using Tailwind): https://tailwindcss.com/docs/preflight
- Teddy Bradford - Modern-Normalise: https://github.com/sindresorhus/modern-normalize/blob/main/modern-normalize.css
```css
/* =========================================
   reset.css — Modern baseline (long-term)
   ========================================= */

/* Box sizing rules */
*, *::before, *::after {
  box-sizing: border-box;
}

/* Prevent iOS text size adjust surprises */
html {
  -webkit-text-size-adjust: 100%;
}

/* Remove default margin on common elements */
body, h1, h2, h3, h4, h5, h6,
p, figure, blockquote, dl, dd {
  margin: 0;
}

/* Core body defaults */
body {
  min-height: 100vh;
  line-height: 1.5;
}

/* Strong/b emphasis fix */
b, strong {
  font-weight: bolder;
}
/* summary display fix (edge-case but legit) If you ever use <details>: */
summary {
  display: list-item;
}

/* Lists: keep normal lists; only remove when you opt-in */
ul[role="list"], ol[role="list"] {
  list-style: none;
  padding: 0;
  margin: 0;
}

/* Smooth scroll for keyboard users */
html:focus-within {
  scroll-behavior: smooth;
}

/* A elements that don't have a class get default styles */
a:not([class]) {
  text-decoration-skip-ink: auto;
}

/* Media defaults */
img, picture {
  max-width: 100%;
  display: block;
}

/* Inherit fonts for form controls */
button, optgroup, input, button, textarea, select {
  font: inherit;
}

/* Tables (practical legacy fix that still helps) */
table {
  border-collapse: collapse;
  border-spacing: 0;
}

/* Reduced motion */
@media (prefers-reduced-motion: reduce) {
  html:focus-within {
    scroll-behavior: auto;
  }

  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
    scroll-behavior: auto !important;
  }
}
```

### **Brief notes (what you need to know)**

- This is essentially **Andy Bell’s modern reset**, plus a **tiny table fix** and an iOS text-size safety line. 
    
- If you use **Tailwind**, don’t use this file globally. Tailwind’s **Preflight** already covers the baseline reset; add only small overrides on top.
