## **Goal**

- Fonts load **before first paint**
    
- No fallback → swap → layout shift
    
- You control exactly what files are used
    

---

## **1) Install the font packages (Fontsource)**

- Example forinter and source-serif-4
```bash
npm i @fontsource-variable/inter @fontsource-variable/source-serif-4
```

---

## **2) Find which** 

## **.woff2**

##  **files you actually use**

  

You only want the **latin wght normal** ones (unless you need italic / extended sets).

  

Typical ones:

- inter-latin-wght-normal.woff2
    
- source-serif-4-latin-wght-normal.woff2
    

  

How to confirm:

- Run dev server
    
- DevTools → **Network** → filter font
    
- Reload → copy the exact .woff2 filenames being requested
    

---

## **3) Copy ONLY those** 

## **.woff2**

##  **files into** 

## **public/fonts/**

  

Example folders:

```code
public/fonts/inter/inter-latin-wght-normal.woff2
public/fonts/source-serif-4/source-serif-4-latin-wght-normal.woff2
```

(You’re copying from node_modules/@fontsource-variable/.../files/...woff2)

---

## **4) Define the fonts with** 

## **@font-face**

##  **(put this in** 

## **src/styles/reset.css**

## **)**

  

Put it at the top of the file (before body styles).

```css
/* ===== Fonts (local woff2 files) ===== */
@font-face {
  font-family: "Inter Variable";
  font-style: normal;
  font-display: swap;
  font-weight: 100 900;
  src: url("/fonts/inter/inter-latin-wght-normal.woff2") format("woff2-variations");
}

@font-face {
  font-family: "Source Serif 4 Variable";
  font-style: normal;
  font-display: swap;
  font-weight: 200 900;
  src: url("/fonts/source-serif-4/source-serif-4-latin-wght-normal.woff2") format("woff2-variations");
}
```

Notes:

- font-family names must match what you use in CSS.
    
- src: path must match the file in public/.
    

---

## **5) Preload the exact font files in`<head>`(BaseLayout)**

```astro
<link
  rel="preload"
  href="/fonts/inter/inter-latin-wght-normal.woff2"
  as="font"
  type="font/woff2"
  crossorigin
/>
<link
  rel="preload"
  href="/fonts/source-serif-4/source-serif-4-latin-wght-normal.woff2"
  as="font"
  type="font/woff2"
  crossorigin
/>
```

This is the critical “no CLS” step.

---

## **6) Apply fonts in your reset (your current pattern is correct)**

```css
body {
  font-family: "Inter Variable", system-ui, -apple-system, "Segoe UI", Roboto, Arial, sans-serif;
  line-height: 1.5;
}

h1, h2, h3, h4, h5, h6 {
  font-family: "Source Serif 4 Variable", Georgia, "Times New Roman", serif;
}
```


---

## **7) Verify it worked (fast checks)**

1. Lighthouse CLS should drop (fonts stop being culprits)
    
2. DevTools → Network → filter font → the requests should be:
    

  

- /fonts/inter/...
    
- /fonts/source-serif-4/...
    

  

Not anything from /_astro/ or node_modules.

---

# **Your “best practices” checklist**

  

Use this every time:

- Install Fontsource package
    
- Identify the exact .woff2 files actually used (Network tab)
    
- Copy only those files to public/fonts/...
    
- Add @font-face in reset.css that points to /fonts/...
    
- Add `<link rel="preload" as="font"> for each font in <head>`
    
- Apply font-family on body + headings
    
- Remove JS/CSS imports from @fontsource-* once you’re using @font-face
    
- Verify font requests come from /fonts/...
