Utilities with tailwind
```css
  
@layer utilities {
  /* ---------- Layout primitives (composition) ---------- */

  /* Vertical rhythm for content blocks (Andy Bell / CUBE pattern) */
  .flow > * + * {
    margin-block-start: var(--flow-space, 1rem);
  }

  /* Centered page wrapper */
  .wrapper {
    margin-inline: auto;
    max-width: 80rem;
    padding-inline: clamp(1rem, 4vw, 2rem);
  }

  /* A “cluster” is better than a generic .flex */
  .cluster {
    display: flex;
    flex-wrap: wrap;
    gap: var(--gap, 1rem);
    align-items: center;
  }

  /* A “stack” is better than a generic .grid for vertical layout */
  .stack {
    display: flex;
    flex-direction: column;
    gap: var(--gap, 1rem);
  }

  /* ---------- Small helpers (utilities) ---------- */

  .list-reset {
    list-style: none;
    padding: 0;
    margin: 0;
  }

  /* Screen-reader only (Tailwind has sr-only, but keep this if you want framework-free portability) */
  .sr-only {
    position: absolute;
    width: 1px;
    height: 1px;
    padding: 0;
    margin: -1px;
    overflow: hidden;
    clip: rect(0, 0, 0, 0);
    white-space: nowrap;
    border: 0;
  }

  /* Optional: show sr-only content when focused (skip links etc.) */
  .sr-only-focusable:active,
  .sr-only-focusable:focus {
    position: static;
    width: auto;
    height: auto;
    margin: 0;
    overflow: visible;
    clip: auto;
    white-space: normal;
  }
}

}
```
