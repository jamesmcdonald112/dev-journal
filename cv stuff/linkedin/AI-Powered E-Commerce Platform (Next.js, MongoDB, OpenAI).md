## Description 
Full-stack AI e-commerce platform built with Next.js, TypeScript, MongoDB, and OpenAI.  
Features include semantic search, RAG Q&A, admin product management, and a complete shopping/cart flow.  
Implements Zod-validated APIs, vector search (MongoDB Atlas), reusable UI components, and a clean feature-based architecture.
