# GitHub - jamesmcdonald112/simple-banking-system-full-stack

## Description
Built with Spring Boot (Java) and React/TypeScript, this project simulates real banking workflows: account creation, login with card/PIN/password, deposits, and transfers. Features Luhn-based card generation, global error handling, REST APIs, and a responsive React UI with modals, validation, and toast notifications.
