## **1. OpenRouter**

  

**Link:** https://openrouter.ai/models?fmt=cards&input_modalities=video

  

**What it is:**

A unified API platform that gives access to many AI models from different providers through one interface.

  

**Key points:**

- Supports multiple input types, including video.
    
- Centralised model marketplace and billing.
    
- Good for comparing and switching between models easily.
    

---

## **2. Mistral AI**

  

**Link:** https://mistral.ai/pricing#api-pricing

  

**What it is:**

A platform offering Mistral’s own high-performance language models with clear pricing tiers.

  

**Key points:**

- Free tier + Pro and enterprise options.
    
- Models for text generation, code tasks, and embeddings.
    
- High-speed, focused models (not a marketplace).
    

---

## **3. Z.AI**
  

**Link:** https://z.ai/subscribe

  

**What it is:**

An AI platform offering coding assistance tools and multimodal model access.

  

**Key points:**

- Subscription-based access.
    
- Token-based pricing for text; separate pricing for video/image tasks.
    
- Aimed at developers needing coding support and media generation tools.
    

---

## **4. MiniMax API**

##  **(Anthropic-style API)**

  

**Link:** https://platform.minimax.io/docs/api-reference/text-anthropic-api

  

**What it is:**

A multi-modal AI API that uses the same request format as Anthropic’s API.

  

**Key points:**

- Supports text, audio, image, video, and other modes.
    
- Easy to integrate if you already understand Anthropic API structures.
    
- Good for projects needing a wide range of media capabilities.
