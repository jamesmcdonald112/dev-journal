## **2) Animation tools (what you actually use)**

  

### **Core**

- [**GSAP**](chatgpt://generic-entity?number=4) **(free)**
    
    **Why:** professional control for hover, scroll, timelines.
    
    **Use cases:** guide attention, emphasize CTAs, smooth reveals.

- [**madewithgsap**](https://madewithgsap.com/get-started) **(PAID)**
    
    **Why:** professional control for hover, scroll, timelines.
    
    **Use cases:** guide attention, emphasize CTAs, smooth reveals.


    

  

### **Optional**

- [**Lottie**](chatgpt://generic-entity?number=5) **(free)**
    
    **What it is:** plays lightweight **JSON vector animations** exported from After Effects.
    
    **When to use:** icons, loaders, small illustrations with motion.
    
    **When not:** large UI transitions or layout changes (use GSAP instead).
