# **1. Install Biome**

  

Run this inside your project:

```bash
npm install --save-dev --save-exact @biomejs/biome
```

---

# **2. Generate the default config**

```bash
npx @biomejs/biome init
```

This creates a **biome.json** file in the root.

---

# **3. Replace biome.json with this EXACT working version**

  

Paste this into biome.json:

```json
{
  "$schema": "https://biomejs.dev/schemas/2.3.7/schema.json",
  "vcs": {
    "enabled": true,
    "clientKind": "git",
    "useIgnoreFile": true
  },
  "formatter": {
    "enabled": true,
    "indentStyle": "space",
    "indentWidth": 2,
    "lineWidth": 100
  },
  "linter": {
    "enabled": true,
    "rules": {
      "recommended": true
    }
  },
  "javascript": {
	  "formatter": {
	    "quoteStyle": "double"
	  }
	},
  "overrides": [
    {
      "includes": ["**/*.astro", "**/*.svelte", "**/*.vue"],
      "linter": {
        "rules": {
          "style": {
            "useConst": "off",
            "useImportType": "off"
          },
          "correctness": {
            "noUnusedVariables": "off",
            "noUnusedImports": "off"
          }
        }
      }
    }
  ]
}
```

### **What this gives you:**

- Formatting (spaces, width 100)
    
- Linting (recommended rules)
    
- .astro/.vue/.svelte support
    
- No false warnings about unused imports inside Astro files
    
- Git ignore integration
    

---

# **4. (Optional) Add JS quote-style preference** (Included above)

  

If you want **double quotes in all JS/TS files**, you _may_ add this block:

```json
"javascript": {
  "formatter": {
    "quoteStyle": "double"
  }
}
```

**This does NOT affect Astro.**

It only changes formatting inside JS/TS and `<script>` sections.

  

If you don’t care → leave it out.

---

# **5. Add scripts to package.json**

  

In your "scripts" section, add these:

```json
"lint": "biome lint --write .",
"lint:fix": "biome check --write .",
"format": "biome format --write .",
"tidy": "biome check --write ."
```

Result:

```json
"scripts": {
  "dev": "astro dev",
  "build": "astro build",
  "preview": "astro preview",
  "lint": "biome lint --write .",
  "lint:fix": "biome check --write .",
  "format": "biome format --write .",
  "tidy": "biome check --write ."
}
```

### **“tidy” =**

  

**lint + auto-fix + organize imports + format**

A full clean-up in one command.

---

# **6. Test your setup**

  

Run:

```
npx @biomejs/biome check --write .
```

If it finishes with no config errors → **Biome is fully working**.
