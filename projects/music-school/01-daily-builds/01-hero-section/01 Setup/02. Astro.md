Docs -  [Astro](https://docs.astro.build/en/install-and-setup/)

### **1. Create the project**

  

Run:

```bash
npm create astro@latest harmony-music-academy
```

Choose:

- **Minimal (empty) template**
    
- **Install dependencies: Yes**
    
- **Initialize git: Yes**
    

  

This created:

- The Astro project folder
    
- The default files
    
- A git repo
