# **Step 1 — Add Tailwind using Astro’s built-in command**

  

Run this inside your project folder:

```bash
npx astro add tailwind
```

You’ll be prompted to confirm:

- Install Tailwind? → **Yes**
    
- Create global.css? → **Yes**
    
- Update astro.config.mjs? → **Yes**
    

  

Astro automatically performs:

  

### **✔ Installs:**

- tailwindcss
    
- @tailwindcss/vite
    

  

### **Updates your config (astro.config.mjs)**

```bash
import tailwindcss from '@tailwindcss/vite';

export default defineConfig({
  vite: {
    plugins: [tailwindcss()]
  }
});
```

### **✔ Creates the stylesheet:**

```
src/styles/global.css
```

with:

```
@import "tailwindcss";
```

This file contains Tailwind’s base layers.

# **Step 2 — Create your layout file and import Tailwind**

  

Astro does not create the layout for you.

You must create it based on the Layouts documentation.

  

Create:

```
src/layouts/Layout.astro
```

Add this exact content:

```ts
---
import '../styles/global.css';

interface Props {
  title: string;
  description: string;
}

const { title, description } = Astro.props;
---

<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <meta name="description" content={description} />
    <title>{title}</title>
  </head>
  <body class="antialiased">
    <slot />
  </body>
</html>
```

This does three important things:

- Imports Tailwind globally
    
- Defines your page shell
    
- Provides <slot /> for wrapping every page
    

  

(From docs: https://docs.astro.build/en/basics/layouts/)
