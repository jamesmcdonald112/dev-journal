## Configure Astro SDK


Sentry's integration with [Astro](https://astro.build/) supports Astro 3.0.0 and above.

In this quick guide you'll use the **astro** CLI to set up Sentry with separate configuration files for client and server-side initialization:

Error Monitoring

Logs

Metrics

Session Replay

Tracing

---

#### Install

Install the `@sentry/astro` package with the `astro` CLI:

bash

```bash
npx astro add @sentry/astro
```

#### Configure SDK

Copy DSN

Configure the Sentry integration in your `astro.config.mjs` file:

astro.config.mjs

```javascript
import { defineConfig } from "astro/config";
import sentry from "@sentry/astro";

export default defineConfig({
  integrations: [
    sentry({
      project: "javascript-astro",
      org: "james-mcdoanld",
      authToken: process.env.SENTRY_AUTH_TOKEN,
    }),
  ],
});
```

Create a `sentry.client.config.js` file in the root of your project to configure the client-side SDK:

sentry.client.config.js

```javascript
import * as Sentry from "@sentry/astro";

Sentry.init({
  dsn: "https://ca66e5fae13929e4cdaacce4183e79d4@o4510425283231744.ingest.de.sentry.io/4510425285787728",
  // Adds request headers and IP for users, for more info visit:
  // https://docs.sentry.io/platforms/javascript/guides/astro/configuration/options/#sendDefaultPii
  sendDefaultPii: true,
});
```

Create a `sentry.server.config.js` file in the root of your project to configure the server-side SDK:

sentry.server.config.js

```javascript
import * as Sentry from "@sentry/astro";

Sentry.init({
  dsn: "https://ca66e5fae13929e4cdaacce4183e79d4@o4510425283231744.ingest.de.sentry.io/4510425285787728",
  // Adds request headers and IP for users, for more info visit:
  // https://docs.sentry.io/platforms/javascript/guides/astro/configuration/options/#sendDefaultPii
  sendDefaultPii: true,
});
```

Add your Sentry auth token to the `SENTRY_AUTH_TOKEN` environment variable:

bash

```bash
SENTRY_AUTH_TOKEN=Click to generate token
```

#### Verify

Then throw a test error anywhere in your app, so you can test that everything is working:

Astro

```html
<!-- your-page.astro -->
---
---
<button id="error-button">Throw test error</button>
<script>
  function handleClick () {
    throw new Error('This is a test error');
  }
  document.querySelector("#error-button").addEventListener("click", handleClick);
</script>
```
