### **Biome handles everything (JS, TS, JSON, CSS, config files…)**

  

### **Prettier handles ONLY .astro files**

  

Why?

- Biome **does not support** **.astro** **syntax**
    
- Prettier is the **only stable formatter for Astro**
    

  

So we configured both tools so they **do not overlap**.

# **✅** 

# **2. We told VSCode which formatter to use**

  

Your .vscode/settings.json:

```json
{
  "cSpell.words": ["gsap"],
  "editor.formatOnSave": true,
  "editor.defaultFormatter": "biomejs.biome",
  "[astro]": {
    "editor.defaultFormatter": "esbenp.prettier-vscode"
  }
}
```

Meaning:

- **All files → Biome formats**
    
- **Astro files → Prettier formats**
    

  

This matches the official docs:

- Biome docs: _Biome should be the default formatter_
    
- Astro docs: _Prettier must be used for_ _.astro_

# **✅** 

# **3. We configured Biome so it tracks everything EXCEPT Astro**

  

Your biome.json:

```json
"files": {
  "ignoreUnknown": false,
  "includes": ["**", "!**/*.astro"]
},
"formatter": {
  "enabled": true,
  "indentStyle": "tab"
},
"linter": {
  "enabled": true,
  "rules": { "recommended": true }
},
"assist": {
  "enabled": true,
  "actions": { "source": { "organizeImports": "on" } }
}
```

Effect:

- Biome **never touches** **.astro**
    
- Biome formats/lints **every other file**
    
- Biome auto-sorts imports
    
- Biome supports Tailwind directives (@plugin, etc.)

# **4. We built the correct scripts in package.json**

  

Your scripts:

```json
"format": "biome format --write .",
"lint": "biome check --write .",
"format:astro": "prettier --write \"**/*.astro\"",
"tidy": "npm run lint && npm run format:astro"
```

Meaning:

  

### **npm run lint**

- Biome lints **everything except Astro**
    
- Applies **safe** fixes automatically
    

  

### **npm run format**

- Biome formats **everything except Astro**
    

  

### **npm run format:astro**

- Prettier formats **only .astro**
    

  

### **npm run tidy**

  

Runs all cleaning:

1. Lint + fix (Biome)
    
2. Format .astro (Prettier)
    

  

This is the “one button clean my whole repo” command.
