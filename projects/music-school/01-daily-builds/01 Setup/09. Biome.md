# **1. Install the required tools**

  

Biome (formats everything except .astro):

```bash
npm i -D -E @biomejs/biome
```

(installed previously) Prettier (formats ONLY .astro files):

```bash
npm i -D -E prettier prettier-plugin-astro prettier-plugin-tailwindcss
```

---

# **2. Why we use BOTH**

  

### **Biome = JS, TS, JSON, CSS, config files**

  

### **Prettier = Astro files only**

  

Because:

- Biome **cannot** parse .astro
    
- Prettier is the **official formatter for Astro**
    
- Using both gives full coverage with **zero formatter conflicts**
    

---

# **3. VSCode configuration**

  

Add to .vscode/settings.json:

```json
{
  "cSpell.words": ["gsap"],
  "editor.formatOnSave": true,

  // Biome: default formatter for everything
  "editor.defaultFormatter": "biomejs.biome",

  // Prettier: ONLY for .astro
  "[astro]": {
    "editor.defaultFormatter": "esbenp.prettier-vscode"
  }
}
```

Meaning:

- Every file uses Biome **except** .astro
    
- .astro always uses Prettier

# **4. biome.json (configured for Tailwind v4)**

  

Your biome.json **must** include Tailwind support:

```json
{
  "$schema": "https://biomejs.dev/schemas/2.3.7/schema.json",
  "vcs": {
    "enabled": true,
    "clientKind": "git",
    "useIgnoreFile": true
  },
  "files": {
    "ignoreUnknown": false,
    "includes": ["**", "!**/*.astro"]
  },
  "formatter": {
    "enabled": true,
    "indentStyle": "tab"
  },
  "linter": {
    "enabled": true,
    "rules": {
      "recommended": true
    }
  },
  "javascript": {
    "formatter": {
      "quoteStyle": "double"
    }
  },
  "css": {
    "parser": {
      "tailwindDirectives": true
    }
  },
  "assist": {
    "enabled": true,
    "actions": {
      "source": {
        "organizeImports": "on"
      }
    }
  }
}
```

Effects:

- Biome **ignores** .astro
    
- Biome formats/lints **everything else**
    
- Tailwind v4 directives (@plugin, @import) **no longer break**
    
- Import sorting enabled
    
- Safe fixes applied automatically

# **5. package.json scripts**

  

Add:

```json
"scripts": {
  "lint": "biome check --write .",
  "format": "biome format --write .",
  "format:astro": "prettier --write \"**/*.astro\"",
  "tidy": "npm run lint && npm run format:astro"
}
```

Meaning:

  

### **npm run lint**

- Lints & fixes all non-Astro files (Biome)
    

  

### **npm run format**

- Formats all non-Astro files (Biome)
    

  

### **npm run format:astro**

- Formats ONLY .astro files (Prettier)
    

  

### **npm run tidy**

  

Your all-in-one cleanup command:

1. Lint + fix everything except Astro (Biome)
    
2. Format Astro files (Prettier)
