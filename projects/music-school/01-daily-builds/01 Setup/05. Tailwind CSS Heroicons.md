# **3. How to style SVGs with Tailwind**

  

Add classes to the SVG:

```html
<svg class="w-5 h-5 text-gray-500" ...></svg>
```

Use modifiers:

- w-4 h-4
    
- w-6 h-6
    
- text-green-600
    
- hover:text-green-700
    

  

Example:

```html
<button class="flex items-center gap-2 text-lg font-medium">
  <svg class="w-5 h-5 text-green-500" ...></svg>
  Contact Us
</button>
```
