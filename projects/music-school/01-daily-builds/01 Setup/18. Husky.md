
# **1. Install Husky**

```
npm install --save-dev husky
```

---

# **2. Initialise Husky (creates .husky folder + default hook)**

```
npx husky init
```

This does **two things**:

1. Creates .husky/pre-commit
    
2. Adds "prepare": "husky" to package.json
    

  

You now have a working Git hook.

---

# **3. Check generated pre-commit hook**

  

.husky/pre-commit contains:

```
npm test
```

Meaning: every commit triggers your test suite.

---

# **4. Add additional hooks (optional)**

  

Use Husky’s official API:

  

### **Example: commit-msg hook (for commitlint)**

```
echo "npx --no -- commitlint --edit \$1" > .husky/commit-msg
```

This creates:

```
.husky/commit-msg
└── npx --no -- commitlint --edit $1
```

This ensures commit messages follow Conventional Commits.

---

# **5. Running scripts inside Husky**

  

Husky executes **POSIX shell**, so you can run:

- npm run ...
    
- npx ...
    
- direct shell scripting
    

  

Example custom script:

```
# .husky/pre-commit
npm run lint
npm run test
```

Or raw shell:

```
# .husky/pre-commit
echo "Running checks…"
```

---

# **6. Disable Husky (when needed)**

  

**Single command:**

```
git commit -m "msg" -n
```

**Temporary global disable:**

```
export HUSKY=0
```

**Re-enable:**

```
unset HUSKY
```
