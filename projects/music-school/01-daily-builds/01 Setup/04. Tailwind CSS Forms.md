**Reason:**

It completely overrides all default form styles (border-radius, padding, borders, outlines, backgrounds, etc.).

This made it extremely hard to style my own inputs without using !important everywhere.

  

My form components (.form-input, .form-textarea) work correctly only **after removing** the Forms plugin.

## **3. What the plugin actually does**

  

It provides a **basic form reset** so that form elements become:

- visually consistent across browsers
    
- easy to style using Tailwind utilities
    
- easy to customise without writing custom CSS
    

  

It normalises these elements:

- all input[type=...]
    
- select, multiple selects
    
- checkbox
    
- radio
    
- textarea

  strategy: "class";
