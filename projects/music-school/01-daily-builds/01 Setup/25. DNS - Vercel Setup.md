## **Key clarifications (important)**

  

### **Why does root use A and WWW use CNAME?**

- Root **must** use A (DNS limitation)
    
- Subdomains **should** use CNAME
    
- This is normal, not special to Vercel
    

---

### **Why delete instead of ‚Äúadjust‚Äù records?**

  

Because:

- You can‚Äôt have **A + CNAME** for the same name
    
- Old records conflict with new ones
    
- DNS must be unambiguous
    

---

### **Does Vercel only show these once?**

- Vercel shows them **until DNS is correct**
    
- You can always re-check them later
    
- They are not ‚Äúone-time secrets‚Äù
