Docs - https://docs.astro.build/en/guides/integrations-guide/react/

I installed React in this Astro project **only because I need to use React components**, such as:

- **Sonner toast notifications**
    
- **shadcn/ui components**
    
- **Any custom** **.tsx** **UI component I build**
    

  

Astro cannot render .tsx files without the React integration.

The fix is installing React:

```
npx astro add react
```

This command:

- Installs react, react-dom, and @astrojs/react
    
- Updates astro.config.mjs
    
- Enables React islands inside Astro pages
    

  

After this, I can safely use:

```
<ToasterClient client:load />
```

and any other React component alongside my .astro files.
