# **✅** 

# **Step 1 — Add Tailwind using Astro’s built-in command**

  

Run this inside your project folder:

```bash
npx astro add tailwind
```

Choose **Yes** for all prompts:

- Install Tailwind? → **Yes**
    
- Create global.css? → **Yes**
    
- Update astro.config.mjs? → **Yes**
    

  

Astro automatically performs:

  

### **✔ Installs:**

- tailwindcss
    
- @tailwindcss/vite
    

  

### Updates your

### **astro.config.mjs**

### **:**

```ts
import tailwindcss from '@tailwindcss/vite';

export default defineConfig({
  vite: {
    plugins: [tailwindcss()]
  }
});
```

### **✔ Creates the stylesheet:**

```
src/styles/global.css
```

with:

```css
@import "tailwindcss";
```

This loads Tailwind’s base layers.

# **✅** 

# **Step 2 — Create your layout file and import Tailwind**

  

Astro requires **you** to create your own layout.

This is where global Tailwind styling is imported.

  

Create:

```
src/layouts/Layout.astro
```

Add this exact content:

```ts
---
import '../styles/global.css';

interface Props {
  title: string;
  description: string;
}

const { title, description } = Astro.props;
---

<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="description" content={description} />
    <title>{title}</title>
  </head>
  <body class="antialiased">
    <slot />
  </body>
</html>
```

This layout:

- Imports Tailwind globally
    
- Sets the HTML document shell
    
- Handles title + description
    
- Wraps page content using <slot />
    

  

(From docs: https://docs.astro.build/en/basics/layouts/)

---

# **✅** 

# **Step 3 — Wrap your pages with the layout**

  

Here’s your actual working index.astro:

```ts
---
import Layout from '../layouts/Layout.astro';
---

<Layout 
  title="Harmony Music Academy"
  description="Dublin’s premier music school"
>
  <h1 class="text-4xl font-bold text-green-400">
    Hello g
  </h1>
</Layout>
```

This ensures:

- The entire page is inside your layout
    
- Tailwind classes work
    
- Title and description appear in `<head>`
    
- `<slot />` receives your content
