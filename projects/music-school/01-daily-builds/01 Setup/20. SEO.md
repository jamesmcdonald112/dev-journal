# **STEP 1 â€” Metadata / Head Setup (the foundation)**

  

This is the part every SEO strategy depends on.

  

Astro does _not_ give you automatic SEO.

You must create a **reusable <Head> component** that:

- sets default title
    
- sets default description
    
- allows overrides per page
    
- injects canonical URL
    
- injects OpenGraph
    
- injects Twitter metadata
    
- works with JSON-LD later

# **1.1 â€” Create a global Head component**

  

Create this file:

```
src/components/seo/Head.astro
```

Paste this (safe, minimal version â€” we expand later):

```
---
const {
  title = "Harmony Music Academy",
  description = "Professional music lessons and courses.",
  canonical = Astro.url.href,
  image = "/default-og.png",
} = Astro.props;
---

<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1" />

  <title>{title}</title>
  <meta name="description" content={description} />

  <!-- Canonical -->
  <link rel="canonical" href={canonical} />

  <!-- OpenGraph -->
  <meta property="og:title" content={title} />
  <meta property="og:description" content={description} />
  <meta property="og:image" content={image} />
  <meta property="og:url" content={canonical} />
  <meta property="og:type" content="website" />

  <!-- Twitter -->
  <meta name="twitter:card" content="summary_large_image" />
  <meta name="twitter:title" content={title} />
  <meta name="twitter:description" content={description} />
  <meta name="twitter:image" content={image} />
</head>
```
