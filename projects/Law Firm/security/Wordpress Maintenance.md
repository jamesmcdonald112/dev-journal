# What WordPress attackers can do —** 

# **in plain English**

  

If a WordPress site is insecure, attackers can:

  

### **• Inject malware or spam pages**

  

They add hidden pages to the site that sell scams, illegal products, porn, etc.

  

### **• Install phishing pages**

  

Fake login pages to steal passwords from visitors.

  

### **• Redirect visitors to scam sites**

  

User clicks the solicitor’s site → gets sent to a crypto scam.

  

### **• Scrape or steal form submissions**

  

They can access stored messages:

Names, emails, phone numbers, legal consultation details.

  

### **• Access the admin area**

  

Change content, delete pages, or deface the site.

  

### **• Install crypto-miners**

  

The website secretly mines cryptocurrency using visitors’ computers.

  

### **• Use the site as part of a botnet**

  

The hacked site is used to attack other servers.
