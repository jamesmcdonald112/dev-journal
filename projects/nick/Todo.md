# **3. Moving Their Existing Domain to the New Site**

  

You **do not move the domain** anywhere.

You simply **change DNS settings** to point the domain to Vercel or Netlify.

  

### **Steps:**

1. Log in to their domain registrar (GoDaddy, Blacknight, etc.)
    
2. Remove the old DNS records
    
3. Add the new DNS records from Vercel/Netlify
    
4. Wait for DNS to update (30 minutes ‚Äì 24 hours)
    

  

**The domain stays with them.**

**The website now lives on modern hosting.**
