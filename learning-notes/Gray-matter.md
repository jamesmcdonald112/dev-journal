# Gray-matter

## What I Learned
It is a `Markdown frontmatter parser`, meaning it takes the .md file and returns
- the metadata (YAML/JSON frontmatter)
- the content (the Markdown body)

## Example / Code Snippet
```js
const { data, content } = matter(markdownFile);
```
