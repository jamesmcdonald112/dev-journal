# AMI (Amazon Machine Image)

## What I Learned
An AMI is a template that defines an EC2 instance’s OS, software, and configuration.

## Key Questions
- How do AMIs differ from snapshots or containers?
- When should you create a custom AMI?
- How does AMI versioning help in deployments?
## Apply It
- Launch two identical EC2 instances from the same AMI.
- Create a custom AMI from a configured instance.
