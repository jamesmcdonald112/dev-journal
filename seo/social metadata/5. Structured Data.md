This is not optional for serious local SEO.

  

It tells Google:

- You are a LegalService
    
- Located in Kilkenny
    
- Phone number
    
- Address
    

  

Without it:

Google has to guess.

  

With it:

You improve chances of:

- Local pack ranking
    
- Rich results
    
- Business panel
    

---

## **Where to Learn It**

  

Search:

- schema.org LegalService
    
- Google structured data testing tool
    
- Google Rich Results Test


You implement it as:

```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "LegalService",
  "name": "Mary Molloy Solicitors",
  "url": "https://marymolloysolicitors.ie",
  "telephone": "+353567765829",
  "address": {
    "@type": "PostalAddress",
    "streetAddress": "2 Rose Inn St",
    "addressLocality": "Kilkenny",
    "postalCode": "R95 W58D",
    "addressCountry": "IE"
  }
}
</script>
```

Test it in:

Google Rich Results Test.
